*Examples*

Example #1 uopz_restore() example

<?php >
    uopz_backup("fgets");
    uopz_function("fgets", function(){
        return true;
    });
    var_dump(fgets());
    uopz_restore('fgets');
    fgets();
<?>

The above example will output something similar to:
