==============================================================================
*Description*

    object http_persistent_handles_count ( void )~

List statistics about persistent handles usage.

==============================================================================
*Parameters*

==============================================================================
*Return* *Values*

Returns persistent handles statistics as stdClass object on success or FALSE
on failure.

==============================================================================

<?php >

<?>
