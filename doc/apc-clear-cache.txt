==============================================================================
*Description*

    bool apc_clear_cache ([ string $cache_type = "" ] )~

Clears the user/system cache.

==============================================================================
*Parameters*

{cache_type}

    If {cache_type} is "user", the user cache will be cleared; otherwise, the
    system cache (cached files) will be cleared.

==============================================================================
*Return* *Values*

Returns TRUE always
