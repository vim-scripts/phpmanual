==============================================================================
*Description*

    bool fdf_get_ap ( resource $fdf_document , string $field , int $face ,~
    string $filename )~

Gets the appearance of a {field} (i.e. the value of the /AP key) and stores it
in a file.

==============================================================================
*Parameters*

{fdf_document}

    The FDF document handle, returned by |fdf_create|, |fdf_open| or 
    |fdf_open_string|.

{field}

{face}

    The possible values are FDFNormalAP, FDFRolloverAP and FDFDownAP.

{filename}

    The appearance will be stored in this parameter.

==============================================================================
*Return* *Values*

Returns TRUE on success or FALSE on failure.
