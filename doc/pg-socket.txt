==============================================================================
*Description*

    resource pg_socket ( resource $connection )~

pg_socket() returns a read only resource corresponding to the socket
underlying the given PostgreSQL connection.

Warning

This function is currently not documented; only its argument list is
available.

==============================================================================
*Parameters*

{connection}

    PostgreSQL database connection resource.

==============================================================================
*Return* *Values*

A socket resource on success or FALSE on failure.
