==============================================================================
*Description*

    string stream_socket_get_name ( resource $handle , bool $want_peer )~

Returns the local or remote name of a given socket connection.

==============================================================================
*Parameters*

{handle}

    The socket to get the name of.

{want_peer}

    If set to TRUE the remote socket name will be returned, if set to FALSE
    the local socket name will be returned.

==============================================================================
*Return* *Values*

The name of the socket.
