{data}

*Examples*

Example #1 eio_realpath() example

<?php >
    var_dump(getcwd());

    function my_realpath_allback($data, $result) {
        var_dump($result);
    }

    eio_realpath("../", EIO_PRI_DEFAULT, "my_realpath_allback");
    eio_event_loop();
<?>

The above example will output something similar to:

string(12) "/home/ruslan"
string(5) "/home"
