==============================================================================
*Description*

    void yaz_set_option ( resource $id , string $name , string $value )~
    void yaz_set_option ( resource $id , array $options )~

Sets one or more options on the given connection.

==============================================================================
*Parameters*

{id}

    The connection resource returned by |yaz_connect|.

{name} or {options}

    May be either a string or an array.

    If given as a string, this will be the name of the option to set. You'll
    need to give it's {value}.

    If given as an array, this will be an associative array (option name =>
    option value).

                          PHP/YAZ Connection Options
            Name                              Description
    implementationName    implementation name of server
    implementationVersion implementation version of server
    implementationId      implementation ID of server

    preferredRecordSyntax is used. Setting this option is equivalent to using
                          function |yaz_syntax|

    start                 start value 1. Setting this option in combination

    count                 maximum number of records to be retrieved via |

    elementSetName        element-set-name for retrieval. Setting this option

{value}

    The new value of the option. Use this only if the previous argument is a
    string.

==============================================================================
*Return* *Values*

No value is returned.
