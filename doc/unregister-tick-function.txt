==============================================================================
*Description*

    void unregister_tick_function ( string $function_name )~

De-registers the function named by {function_name} so it is no longer executed
when a tick is called.

==============================================================================
*Parameters*

{function_name}

    The function name, as a string.

==============================================================================
*Return* *Values*

No value is returned.
