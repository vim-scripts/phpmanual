filter_list

(PHP 5 >= 5.2.0)

filter_list â€” Returns a list of all supported filters

==============================================================================
*Description*

    array filter_list ( void )~

==============================================================================
*Return* *Values*

Returns an array of names of all supported filters, empty array if there are
no such filters. Indexes of this array are not filter IDs, they can be
obtained with |filter_id| from a name instead.

==============================================================================
*Examples*

Example #1 A filter_list() example

<?php >
    print_r(filter_list());
<?>

The above example will output something similar to:

Array
