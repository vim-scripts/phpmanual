==============================================================================
*Description*

    int udm_hash32 ( resource $agent , string $str )~

udm_hash32() will take a string {str} and return a quite unique 32-bit hash
number from it.

==============================================================================
*Parameters*

{agent}

    A link to Agent, received after call to |udm_alloc_agent|.

{str}

    The input string.

==============================================================================
*Return* *Values*

Returns a 32-bit hash number.
