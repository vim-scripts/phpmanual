==============================================================================
*Description*

    bool ifx_byteasvarchar ( int $mode )~

Sets the default byte mode for all select-queries.

==============================================================================
*Parameters*

{mode}

    Mode "0" will return a blob id, and mode "1" will return a varchar with
    text content.

==============================================================================
*Return* *Values*

Returns TRUE on success or FALSE on failure.
