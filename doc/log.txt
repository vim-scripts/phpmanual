==============================================================================
*Description*

    float log ( float $arg [, float $base = M_E ] )~

If the optional {base} parameter is specified, log() returns log[base] {arg},
otherwise log() returns the natural logarithm of {arg}.

==============================================================================
*Parameters*

{arg}

    The value to calculate the logarithm for

{base}

    The optional logarithmic base to use (defaults to 'e' and so to the
    natural logarithm).

==============================================================================
*Return* *Values*

The logarithm of {arg} to {base}, if given, or the natural logarithm.

==============================================================================
*Changelog*

Version                              Description
        The optional parameter {base} became available. For older versions you
4.3.0   can calculate the logarithm in base b of a number n, but using the
        mathematical identity: log[b](n) = log(n)/log(b), where log is the
        neperian (or natural) logarithm.
